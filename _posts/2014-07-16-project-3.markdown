- Built a machine learning model to classify fetal health conditions from cardiotocography (CTG) data, focusing on reducing misclassification of high-risk (Pathologic) cases  
- Applied data preprocessing, feature engineering, PCA, and SMOTE to address class imbalance and improve model learning  
- Trained and tuned multiple models, identifying Gradient Boosting as the top performer  
- Achieved ~95% accuracy with strong recall for Pathologic cases, minimizing critical false negatives
