description: This project developed and evaluated machine learning models for fetal health classification using cardiotocography (CTG) data. The workflow included preprocessing, feature engineering, PCA, and SMOTE to address class imbalance and improve model learning. Multiple models were trained and compared, with Gradient Boosting delivering the best performance. The final model achieved about 95% accuracy while maintaining strong recall for Pathologic cases, helping reduce critical false negatives in high-risk predictions.
